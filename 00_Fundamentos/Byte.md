Quando associamos um conjunto de 8 [[Bit]], então damos o nome de [[Byte]]

Um byte é a unidade básica de armazenamento e transmissão de dados em computadores. Ele consiste em 8 bits (0s ou 1s), que podem representar:

- Números (0 a 255)
- Caracteres (letras, números, símbolos)
- Endereços (MAC, IP)
- Outros tipos de dados

Aqui estão algumas utilidades do byte:

1. Endereço MAC: 6 bytes (48 bits) identificam dispositivos de rede.
Ex.: 00:11:22:33:44:55

2. Endereço IP: 4 bytes (32 bits) identificam dispositivos em redes IP.
Ex.: 255.255.255.255

3. Armazenamento de dados: arquivos, imagens, vídeos, texto.
Ex.: 1 KB = 1.024 bytes

4. Transmissão de dados: velocidade medida em bytes/segundo.
Ex.: 1 Mbps = 1.048.576 bytes/segundo

5. Criptografia: bytes são utilizados para representar chaves e dados criptografados.

6. Programação: bytes são utilizados para representar variáveis, constantes e instruções.

Tamanho de unidades de dados:

- Bit (b): 1
- Byte (B): 8 bits
- Kilobyte (KB): 1.024 bytes
- Megabyte (MB): 1.048.576 bytes
- Gigabyte (GB): 1.073.741.824 bytes
- Terabyte (TB): 1.099.511.627.776 bytes

*Aqui estão algumas conversões úteis:_

- 1 byte = 8 bits
- 1 KB = 1.024 bytes
- 1 MB = 1.024 KB
- 1 GB = 1.024 MB
