A [[Camada Interface de Rede| camada de Interface de rede]] transmite de forma transparente os [[pacotes]] recebidos da [[camada de rede]]. Do ponto de vista da [[Camada de Rede|camada de rede]], a [[Camada Interface de Rede|camada de interface]] de rede é uma linha sem erros. Para dar suporte a vários [[protocolos]] da [[camada de rede]], a camada de interface de rede adiciona um ID às informações de controle de um quadro para identificar um [[protocolo]] específico da [[camada de rede]]. Com esse ID, o nó receptor pode enviar corretamente o quadro para o protocolo específico da camada de rede.

Os padrões da camada de interface de rede do IEEE são "os padrões de [[WLAN]]]] mais amplamente usados". Eles são chamados de padrões [[IEEE 802]].
• O 802.1 descreve os problemas básicos da [[WLAN]]]]. Por exemplo, o 802.1d descreve o [[Spanning Tree Protocol]] (STP).
• O grupo 802.2 define a subcamada [[LLC]].
• O grupo 802.3 define os padrões da subcamada [[Subcamada MAC]], como o Carrier Sense Multiple Access with Collision Detection (CSMA/CD).
• O grupo 802.4 define os padrões de [[token bus]].
• O grupo 802.5 define padrões de [[token ring]], que são os mesmos definidos pelo grupo de token ring da IBM.
