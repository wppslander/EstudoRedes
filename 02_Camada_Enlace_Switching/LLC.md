O [[IEEE 802|IEEE]] divide a camada de interface de rede de uma [[WLAN]]]] em duas subcamadas: a subcamada de controle de link lógico ([[LLC]]) e a subcamada de controle de acesso ao meio ([[Subcamada MAC]]). A subcamada [[LLC]] executa o controle de fluxo independente de hardware e o controle de erros na camada de interface de rede. A [[Subcamada MAC]], abaixo da subcamada [[LLC]], atua como uma interface entre a subcamada [[LLC]] e a [[camada física]]. Diferentes tipos de [[WLAN]]]] usam o mesmo padrão de subcamada [[LLC]], mas diferentes padrões de subcamada [[Subcamada MAC]].

A camada de interface de rede encapsula e identifica os dados recebidos de sua camada superior, que é implementada pela subcamada LLC.

A subcamada [[LLC]] multiplexa os protocolos adicionando um cabeçalho 802.2 LLC ao pacote que a camada de rede fornece. Para identificar os protocolos da camada de rede, a subcamada [[LLC]] usa pontos de acesso de serviço ([[SAP]]s). Dois [[SAP]]s, o [[SAP]] de origem ([[SSAP]]) e o SAP de destino ([[DSAP]]), são usados para identificar os protocolos da camada de rede do remetente e do destinatário, respectivamente. O campo [[SSAP]] e o campo [[DSAP]] têm, cada um, um [[Byte]] de comprimento, dos quais apenas seis [[Bit]]s são usados para identificar o protocolo da camada superior. Portanto, o [[SAP]] pode identificar não mais do que 32 [[protocolos]]. Para fazer com que a subcamada 802.2 [[LLC]] suporte mais protocolos da camada superior, o IEEE emitiu o 802.2 [[Subnetwork]] Access Protocol ([[SNAP]]). O IEEE 802.2 [[SNAP]] estende o cabeçalho [[LLC]] para encapsular dados da camada superior. Com o IEEE 802.2 SNAP, o valor [[SAP]] é definido como hexadecimal AA, e um campo de tipo de dois bytes é adicionado.
 

A camada de [[Camada Interface de Rede]] controla o acesso ao meio de transmissão, lida com contenções de canal, identifica e endereça estações e verifica a integridade dos quadros de entrada na subcamada [[Subcamada MAC]]. O IEEE define diferentes padrões de subcamada [[Subcamada MAC]] para diferentes meios: 802.3 para [[Ethernet]], 802.4 para [[token bus]] e 802.5 para [[token ring]]. Além disso, a camada [[Subcamada MAC]] é responsável pela verificação da integridade dos quadros de dados recebidos.
