Uma **máquina virtual (VM)** é um ambiente computacional que simula um computador físico dentro de um software. Ela permite que você execute um sistema operacional (como Windows, Linux, etc.) e aplicativos como se fosse um computador independente, mas na verdade está rodando em cima do hardware e do sistema operacional de um servidor ou computador [[host]]]].

### Principais Características:

- **Isolamento**: Cada VM funciona de forma independente das outras, isolando o sistema operacional e os aplicativos instalados nela.
- **Recursos Virtuais**: A VM simula recursos como CPU, memória, armazenamento e rede, que são fornecidos pelo hardware físico do [[host]]]].
- **Flexibilidade**: É possível executar múltiplas VMs em um único servidor físico, cada uma com seu próprio sistema operacional e configuração.

### Vantagens das Máquinas Virtuais:

- **Eficiência no uso de hardware**: Permite utilizar melhor os recursos de um servidor físico, executando várias VMs em uma única máquina.
- **Facilidade de backup e migração**: Uma VM pode ser movida ou copiada facilmente entre servidores.
- **Testes e Desenvolvimento**: Ideal para criar ambientes de teste isolados sem precisar de hardware adicional.
